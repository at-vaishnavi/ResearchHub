# ResearchHub - AngularJS Project

ResearchHub is a web-based project built using **AngularJS** for managing and viewing research publications. This project demonstrates dynamic content rendering, navigation, and basic plagiarism detection features. It allows users to view research publications, check plagiarism, and highlights AI-generated content. The application is fully responsive with AngularJS-driven dynamic pages.

## Features
- View research publications with their details.
- Plagiarism detection for papers.
- Highlights AI-generated content.
- Responsive layout with AngularJS-driven dynamic pages.
- Static register, contact, and submission pages (no backend).

## Special Note on Plagiarism Detection
- For demonstration purposes, the paper at the following link is **recognized as AI-generated**:  
[AI-written paper](https://educationaldatamining.org/edm2024/proceedings/2024.EDM-short-papers.55/index.html)  
- All other papers, text content, and submissions are considered **human-written** and were sourced from **Google/online research**.

**Project Structure:**  
Research-Project/  
├─ assets/        # Images used in the project  
├─ css/           # CSS styles  
├─ html/          # HTML pages (home, about, contact, publications, plagiarism, index, register)

**How to Run:**  
Clone the repository using `git clone https://github.com/at-vaishnavi/ResearchHub.git` and open `index.html` in any web browser to start using the project.

**Tech Stack:** HTML, CSS, JavaScript, AngularJS (v1.x), static web application (no backend required).

**Future Improvements:** Add full-fledged plagiarism scanning for multiple papers, dynamic filtering and search for publications.

**Note:** This project is for **educational and demonstration purposes only**. It is not deployed and does not include a backend for submissions. All papers (except the AI example) were sourced from **Google/online research**.
